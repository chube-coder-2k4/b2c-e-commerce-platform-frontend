## 💡 Tips học hiệu quả

### 1. **Đọc → Hiểu → Làm → Sửa**
- Đọc code mẫu
- Hiểu từng dòng (tra Google nếu cần)
- Copy code, chạy thử
- Sửa đổi, xem thay đổi

### 2. **Console.log là vũ khí tốt nhất**
```tsx
console.log('State:', state)
console.log('Props:', props)
console.log('User input:', e.target.value)
```

### 3. **Thay đổi nhỏ, test ngay**
- Không sửa nhiều code cùng lúc
- Mỗi lần sửa 1-2 dòng
- Save → xem kết quả
- Nếu lỗi, Ctrl+Z và thử lại

### 4. **Google là bạn**
- Gặp lỗi? Copy error message → Google
- 99% đã có người gặp trước
- Stack Overflow thường có đáp án

### 5. **Thực hành hàng ngày**
- 30 phút mỗi ngày tốt hơn 3 giờ cuối tuần
- Code mỗi ngày để não nhớ lâu

### 6. **Đọc docs chính thức**
- React: https://react.dev/learn
- Tailwind: https://tailwindcss.com/docs
- TypeScript: https://www.typescriptlang.org/docs

## 🚨 Lỗi thường gặp & Cách fix

### 1. Không thấy thay đổi trên UI
**Nguyên nhân:** Không dùng setState
```tsx
// ❌ Sai
let count = 0
count = count + 1 // UI không update

// ✅ Đúng
const [count, setCount] = useState(0)
setCount(count + 1) // UI tự động update
```

### 2. Component không hiển thị
**Kiểm tra:**
- File `main.tsx` có import đúng?
- Component có `export default`?
- Tên component phải viết hoa chữ cầu đầu

### 3. Tailwind CSS không hoạt động
**Kiểm tra:**
- File `index.css` có `@tailwind` directives?
- Dev server đã restart?

### 4. Import lỗi "@/components/..."
**Giải pháp:** Đã config sẵn, nếu vẫn lỗi restart VSCode
